## 📝 Contributing Reports

### Adding Project Documentation

1. Determine report type (reflection, sustainability, etc.)
2. Write comprehensive document
3. Include data, metrics, and evidence
4. Add to appropriate file in this section
5. Update this README if new report type
6. Submit as PR with description

### Report Standards

✅ **Factual** — Base on actual data and events  
✅ **Clear** — Easy to understand and follow  
✅ **Complete** — Sufficient detail for understanding  
✅ **Dated** — Include dates and timeframes  
✅ **Actionable** — Include recommendations
