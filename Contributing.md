# Contributing Guide

Thank you for contributing to BOSC Community Library.

## Contribution Steps

1. Fork the repository
2. Clone your fork
3. Create a feature branch
4. Make Changes
5. Commit changes
6. Push changes
7. Open a Pull Request

## Branch Naming

feature/new-feature

bugfix/fix-links

refactor/project-cleanup

## Commit Message Format

feat:
fix:
docs:
refactor:

## Rules
-Write clear code
-Update documentation
-Follow respository structure
