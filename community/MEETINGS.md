## 📆 Meeting Schedule

### Regular Meetings

**Community Meetings**
- **Frequency:** 1st Sunday of each month
- **Time:** 2:00 PM UTC
- **Duration:** 1 hour
- **Format:** Open to all community members
- **Purpose:** Project updates, community feedback, Q&A

**Maintainer Meetings**
- **Frequency:** 2nd Wednesday of each month
- **Time:** 1:00 PM UTC
- **Duration:** 45 minutes
- **Participants:** Maintainers + Project Lead
- **Purpose:** Coordination, standards, planning

## 🗣️ How to Participate

### Attend Meetings
1. Check calendar for next meeting date
2. Join link shared in GitHub Discussions
3. Participate in discussion
4. Check notes for action items

### Start a Discussion
1. Open a GitHub Discussion
2. Tag relevant people
3. Clearly state topic & what you'd like feedback on
4. Await community input

### Propose a Decision
1. Submit GitHub Issue with clear proposal
2. Label as "discussion" or "decision-needed"
3. Allow 7 days for community feedback
4. Bring to next meeting for vote

### Report an Action Item
1. If you own an action item, provide updates
2. Report completion or blockers
3. Let team know if needs to be rescheduled
